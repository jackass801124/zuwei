LLMProvider 類別\n\n**職責**：多模型編排與 API 調用\n\n**主要方法**：\n- `initializeContextCache()` - 初始化上下文快取\n- `analyzeChartDeep()` - 使用 Gemini 1.5 Pro 進行深度分析\n- `explainStarQuick()` - 使用 Gemini 1.5 Flash 進行快速解釋\n- `analyzeChartImage()` - 分析命盤圖像\n- `deidentifyUserData()` - 脫敏用戶數據\n- `assessRiskLevel()` - 評估風險等級\n\n**JSON Schema 定義**：\n```typescript\ninterface ZiWeiAnalysisResponse {\n  analysis: {\n    personality: { ... }\n    career_wealth: { ... }\n    relationships: { ... }\n    family: { ... }\n    fortune_predictions: {\n      year_2026: { career, wealth, relationships, health, suggestions }\n      year_2027: { ... }\n      // ... 2028-2030\n      five_year_trend: string\n    }\n  }\n  metadata: {\n    analysis_timestamp: string\n    model_used: string\n    tokens_used: number\n    cache_hit: boolean\n    response_time_ms: number\n    warning_level: 'low' | 'medium' | 'high'\n    risk_assessment: string\n  }\n}\n```\n\n### 2. geminiPromptBuilder.ts - 結構化 Prompt 構建\n\n**職責**：將命盤數據轉換為 Gemini 可理解的結構化 prompt\n\n**主要函數**：\n- `buildGeminiStructuredPrompt()` - 構建完整的結構化 prompt\n- `buildSystemInstruction()` - 構建系統指令\n\n**特點**：\n- 完整的 JSON 結構化數據\n- 詳細的分析維度定義\n- 明確的輸出格式要求\n\n### 3. knowledgeBase.ts - 知識庫管理\n\n**職責**：管理紫微斗數與易經的靜態知識庫\n\n**包含內容**：\n- 14 主星詳細知識\n- 6 吉星與 6 煞星知識\n- 12 宮位含義\n- 四化與五行局知識\n- 易經基礎知識\n\n**快取優勢**：\n- 減少 API 調用中的重複信息\n- 降低 Token 消耗 80%\n- 提升響應速度\n\n### 4. privacyAndMetrics.ts - 隱私與監控\n\n**隱私過濾功能**：\n- `deidentifyChartData()` - 移除敏感個人信息\n- `validatePrivacyCompliance()` - 驗證隱私合規性\n\n**性能監控功能**：\n- `recordMetric()` - 記錄 API 呼叫\n- `getMetrics()` - 獲取所有指標\n- `getSummary()` - 獲取性能摘要\n- `exportAsCSV()` / `exportAsJSON()` - 導出報告\n\n### 5. multimodalInput.ts - 多模態輸入\n\n**功能**：\n- 圖像文件驗證\n- Base64 轉換\n- 圖像調整\n- OCR 文字提取\n- 結果解析與驗證\n- 拖拽上傳支援\n\n### 6. geminiIntegration.ts - 完整整合服務\n\n**職責**：統一的 API 層，整合所有 Gemini 功能\n\n**主要方法**：\n- `analyzeChart()` - 分析命盤（帶重試）\n- `explainStar()` - 解釋星曜（帶重試）\n- `analyzeChartImage()` - 分析圖像（帶重試）\n- `validateAnalysisResponse()` - 驗證回應\n- `formatAnalysisForDisplay()` - 格式化顯示\n\n**重試機制**：\n- 最多重試 3 次\n- 初始延遲 1000ms\n- 指數退避乘數 2\n- 最大延遲 10000ms\n\n### 7. useGeminiAnalysis.ts - React Hooks\n\n**Hook 列表**：\n- `useGeminiAnalysis()` - 主分析 hook\n- `useAnalysisMetrics()` - 性能指標 hook\n- `useWarningLevelStyles()` - 風險等級樣式 hook\n\n## 成本優化策略\n\n### 1. 多模型混合\n\n| 任務 | 模型 | 成本 | 優勢 |\n|------|------|------|------|\n| 深度分析 | Gemini 1.5 Pro | 高 | 精確度高、功能完整 |\n| 快速解釋 | Gemini 1.5 Flash | 低 | 成本低、速度快 |\n\n### 2. 上下文快取\n\n- 靜態知識庫預加載\n- 減少重複 Token 消耗 80%\n- 每次分析只需傳送用戶命盤數據\n\n### 3. Token 優化\n\n- 使用 JSON 格式而非自由文本\n- 結構化數據減少冗餘\n- 知識庫快取避免重複\n\n## 安全性措施\n\n### 1. 隱私保護\n\n- 用戶數據脫敏（移除 ID、郵箱、電話等）\n- 隱私合規性驗證\n- 敏感信息檢測\n\n### 2. 錯誤處理\n\n- 指數退避重試機制\n- API 限流處理\n- 超時管理\n- 詳細的錯誤日誌\n\n### 3. 數據驗證\n\n- JSON Schema 驗證\n- 回應完整性檢查\n- 風險等級評估\n\n## 性能指標\n\n### 追蹤項目\n\n- API 呼叫次數\n- 平均響應時間\n- Token 消耗量\n- 快取命中率\n- 成功率\n- 估計成本\n\n### 導出格式\n\n- CSV 格式（適合分析）\n- JSON 格式（適合集成）\n\n## 使用示例\n\n### 1. 基本分析\n\n```typescript\nconst geminiService = getGeminiIntegration();\nconst result = await geminiService.analyzeChart(chartData);\nconsole.log(result.analysis);\nconsole.log(result.tokensUsed);\nconsole.log(result.responseTimeMs);\n```\n\n### 2. 圖像分析\n\n```typescript\nconst imageBase64 = await MultimodalInputManager.fileToBase64(file);\nconst result = await geminiService.analyzeChartImage(imageBase64);\n```\n\n### 3. 性能監控\n\n```typescript\nconst metrics = geminiService.getPerformanceMetrics();\nconst summary = performanceMonitor.getSummary();\nconsole.log(summary);\n```\n\n## 測試案例\n\n### 測試數據\n\n- 出生日期：1991/11/24 19:30（辛未年十月十九日戌時）\n- 性別：男命\n- 五行局：土五局\n\n### 預期輸出\n\n- ✓ 五個分析維度完整\n- ✓ 2026-2030 年逐年詳細預測\n- ✓ 風險評級準確\n- ✓ JSON 格式有效\n- ✓ Token 消耗在合理範圍\n- ✓ 響應時間 < 30 秒\n\n## 部署檢查清單\n\n- [ ] Google Gemini API Key 配置\n- [ ] 環境變數設置\n- [ ] 知識庫初始化\n- [ ] 隱私過濾測試\n- [ ] 性能監控配置\n- [ ] 錯誤處理驗證\n- [ ] 測試案例通過\n- [ ] 文檔完整\n\n## 後續優化方向\n\n1. **實時流式輸出** - 使用 Gemini 流式 API 提升用戶體驗\n2. **批量分析** - 支援多個命盤的批量分析\n3. **自定義分析** - 允許用戶選擇分析維度\n4. **對比分析** - 支援多個命盤的對比分析\n5. **大限分析** - 加入 10 年大限與小限分析\n6. **PDF 導出** - 支援分析結果導出為 PDF\n7. **多語言支援** - 支援英文、日文等多語言\n8. **API 集成** - 提供公開 API 供第三方集成\n\n## 參考資源\n\n- [Google Gemini API 文檔](https://ai.google.dev/docs)\n- [紫微斗數全書](https://zh.wikipedia.org/wiki/紫微斗數)\n- [周易](https://zh.wikipedia.org/wiki/周易)\n\n---\n\n**文檔版本**：1.0  \n**最後更新**：2026-03-16  \n**作者**：Manus AI 系統架構師\n
